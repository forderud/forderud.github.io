Useful computer programs and libraries

This is a list over some of the tools, programs and libraries I use on a reguar basis. Most are open-source with cross-platform support.

Content creation

  • LyX - latex document processor
  • MiKTeX - latex distribution for windows
  • JabRef - bibtex reference manager
  • Notepad++ - text editor
  • Inkscape - vector graphics editor
  • Paint.NET - simple image processing application
  • Doxygen, (Graphviz) - code documentation generator

Applications/tools

  • Firefox - web browser
  • Foxit reader - PDF reader with annotation support
  • 7-Zip - file compressor
  • Virtual CloneDrive - ISO image mounter
  • Putty - Windows SSH client
  • TortoiseSVN - SVN client with shell integration (SVN monitor)
  • Visual SVN - server for windows
  • WinMerge - diff tool
  • VirtualBox - x86 virtualization

Python

  • Python
  • SciPy/Numpy - numerics packages
  • PyTables - HDF package
  • Matplotlib - plotting package
  • PyUblas - boost ublas integration
