Useful computer programs and libraries

This is a list over some of the tools, programs and libraries I use on a reguar basis. Most are open-source with cross-platform support.

Content creation

  • Notepad++ - text editor
  • Inkscape - vector graphics editor

Applications/tools

  • Chrome - web browser

Python
